# About Me

I am currently pursuing my **M.A. in Economics**, with my thesis focused on the development of a:

## G-SHDI — Geospatial Subnational Human Development Index

This research aims to create a highly granular, pixel-level Human Development Index for Nepal using:

* VIIRS Night-Time Lights
* Terrain & Accessibility Data
* Population Density
* Infrastructure Proxies
* Remote Sensing Indicators
* Machine Learning Models
* Spatial Economic Analysis

The project integrates economics with GIS, remote sensing, and computational modeling to improve development targeting and regional planning.
